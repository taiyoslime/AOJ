# 競プロ万歳

個人的なバックアップのためにOnline Judgeやその他で通したソースコードをあげてます

可読性が最悪なのでソース見るとQoLが低下します。

```
/         : AOJ
/unsolved : AOJ未AC
/atcoder  : Atcoder
/scaffold : 雛形（いらない）
/archives : その他
```
##### 実行環境
- gcc 5.3.0 (Homebrew gcc)
- Mac OS X El Captain 10.11.5

```
$ g++ ${filename} -std=c++11 -Wall -O2
```
