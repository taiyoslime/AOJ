##DP
- 上から降ろしてくるタイプの場合
```

      dp(現在地) = min(現在地,斜め,上)
else
      dp(現在地) = min(現在地,上)
```

- というか現在位置からi+1に配分してく方がミス少ないかな...

##STL
- vectorの重複消去はsort+unique+erase(ゴミ抜き)だが、mapに取込み直しとかすると楽

- set はvectorと比べても死ぬほど重いのでよっぽどのことがない限り
- 積極的にC++11/14使え
